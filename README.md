## 2. 产品概述
### 2.1 产品定位与目标用户
智康 AI 问诊平台是一款面向全年龄段大众、慢性病患者、职场亚健康人群以及有便捷就医需求群体的AI智能医疗健康服务平台，其核心定位为 **“你的 AI 专属健康顾问”**。
本平台旨在通过集成先进的实时在线问诊交互技术、多模态医疗大模型、智能健康数据解析技术以及创新的电子健康档案闭环管理功能，精准解决用户日常健康咨询难、就医排队耗时、慢性病日常管护不便、健康问题无法及时答疑等核心痛点，为用户提供全天候、高效率、专业化、个性化的一站式智能健康问诊与健康管理服务，让优质医疗健康服务触手可及，全方位守护用户身心健康。
### 2.2 核心价值主张  
本平台的核心价值在于构建**在线智能问诊 + 健康管理一站式闭环服务生态：**
1.问诊痛点解决：依托专业医疗知识库与 AI 大模型能力，打造智能问诊交互场景，用户可快速发起健康咨询、症状自查、用药参考，实现问诊 - 诊断 - 建议全流程智能服务，高效解决日常就医咨询、轻症自查的刚需痛点。  
2.全周期健康守护：整合**智能问诊、健康档案管理、慢病跟踪、健康科普**等能力，为用户提供全天候线上健康陪伴，精准生成个性化健康评估与调理建议，实现从即时问诊到长期健康管理的一体化服务。
## 3. 核心功能模块  
### 3.1 AI智能病情分析
| 元素 | 描述 | 交互细节 |
|--------|--------|--------|
| 左侧面板  | **对话创建区**，可查看不同咨询记录  | 用户可在此选择不同对话的历史记录查看，或创建新的咨询对话  |
| 右侧上部面板  | **展示用户与AI的对话内容**  | 用户可在此查看ai进行分析后的指导与建议  |
| 右侧下部面板  | **用户病情输入区**  | 用户可在此输入自己的需求并进行发送给AI，可提交图片、病历本等  |
